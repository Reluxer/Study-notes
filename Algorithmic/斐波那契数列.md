##  描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项。

## 解法

不能用递归，会超时滴。

## code

```cpp
class Solution {
public:
    int Fibonacci(int n){
        if (n==0 || n==1)
            return n;
        int a = 0, b=1, tmp;
        for(int i=2; i<=n; i++){
            tmp = a+b;
            a = b;
            b = tmp;
        }
        return tmp;
    }
};
```
