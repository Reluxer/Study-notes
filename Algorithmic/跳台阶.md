##  描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

## 解法

就是斐波那契数，没什么好说的。

## code

```cpp
class Solution {
public:
    int jumpFloor(int number) {
        int pre=0, cur=1, sum=0;
        for(int i=1; i<=number; ++i){
            sum = cur + pre;
            pre = cur;
            cur = sum;
        }
        return sum;
    }
};
```
